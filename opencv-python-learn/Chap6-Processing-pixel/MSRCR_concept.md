# MSRCR (Multi-Scale Retinex with Color Restoration)

> Retinex 이론은 인간의 시각 시스템이 조명 조건과 무관하게 사물의 본래 색상을 인지하는 방식에 착안한 이미지 처리 기법.
이미지를 '반사율(reflectance)'과 '조명(illumination)' 성분으로 분리하여 조명 성분을 제거하거나 조작함으로써 대비를 향상시킵니다. CLAHE는 대비 향상에 초점을 맞추지만, Retinex는 이미지의 '조명'을 개선하여 더 자연스러운 결과물을 만듭니다.



## MSRCR

> 여러 스케일에서 Retinex를 적용하고, 색상 복원 단계를 추가하여 과도한 색상 왜곡을 방지합니다. 저조도, 안개, 그림자 등 다양한 환경에서 강인한 성능을 보임.

1. **Log 변환**: 이미지를 로그 도메인으로 변환하여 조명과 반사율 성분을 분리하기 쉽게 만듭니다.
`log(R) = log(L) + log(I)`
여기서 R은 원본 이미지, L은 조명 성분, I는 반사율 성분입니다.
2. **조명 성분 추정**: 원본 이미지에 여러 스케일의 가우시안 필터를 적용하여 조명 성분(L)을 추정합니다. 큰 스케일의 필터는 전역적인 조명을, 작은 스케일의 필터는 지역적인 조명을 추정합니다.
3. **반사율 성분 계산**: 원본 이미지에서 추정된 조명 성분을 빼서 반사율 성분(I)을 얻습니다. 즉, log(I) = log(R) - log(L)
4. **다중 스케일 합성**: 각 스케일에서 얻은 반사율 성분들을 가중 평균하여 최종 반사율 성분을 얻습니다.
5. **색상 복원 (Color Restoration - 컬러 이미지의 경우)**: 그레이스케일 이미지에는 해당하지 않지만, 컬러 이미지의 경우 반사율 성분만으로 색상 왜곡이 발생할 수 있어 원본 이미지의 색상 정보를 활용하여 색상 복원 단계를 거칩니다.
6. **Exp 변환 및 정규화**: 로그 변환된 반사율 성분을 다시 원래 스케일로 돌리고, 0-255 범위로 정규화하여 최종 결과 이미지를 얻습니다.

### 장점:
* 이미지의 자연스러운 색상과 대비를 유지하면서 조명 불균형 문제를 해결.
* 명확하고 선명한 결과물을 생성.
* 딥러닝 방법보다는 계산 비용이 적음.

### 단점
* 파라미터 튜닝이 어려울 수 있음.
* 노이즈에 취약할 수 있음.

## 코드 설명
1. `singleScaleRetinex(img, sigma)` 함수:
- 입력 이미지를 `np.float64` 타입으로 변환하고, 로그 변환 시 0이 되는 것을 방지하기 위해 1.0을 더합니다.
- `cv2.GaussianBlur()`를 사용하여 가우시안 필터를 적용하고 조명 성분(L_blur)을 추정합니다. sigma 값이 클수록 넓은 영역의 조명 평균을 나타냅니다.
- 로그 연산을 위해 `L_blur`가 0이 되는 경우를 방지합니다.
- `np.log10(img / L_blur)`를 통해 반사율 성분(로그 도메인)을 계산합니다.
2. `multiScaleRetinex(img, sigma_list)` 함수:
- `sigma_list`에 정의된 여러 스케일(가우시안 필터의 표준 편차)에 대해 `singleScaleRetinex`를 호출합니다.
- 각 스케일에서 얻은 반사율 성분들을 합산한 후 평균하여 최종 다중 스케일 반사율 성분을 얻습니다.
3. `MSRCR_Grayscale(img, sigma_list)` 함수:
- `multiScaleRetinex`를 호출하여 다중 스케일 Retinex 결과(`retinex_result`)를 얻습니다.
- `retinex_result`는 로그 도메인의 값이므로, 이를 0-255 범위의 `uint8` 이미지로 변환하기 위해 Min-Max 정규화를 수행합니다. 이 과정이 그레이스케일 이미지에서의 '색상 복원' 및 '정규화' 역할을 합니다.
4. MSRCR 파라미터 (`sigma_list`) 설정:
- `sigma_list = [15, 80, 250]`는 Retinex에서 조명 성분을 추정할 때 사용할 가우시안 필터의 표준 편차입니다.
  - 작은 `sigma` 값 (예: 15)은 이미지의 지역적인 디테일과 대비를 강조하는 데 기여합니다.
  - 중간 `sigma` 값 (예: 80)은 좀 더 넓은 영역의 조명 변화를 처리합니다.
  - 큰 `sigma` 값 (예: 250)은 이미지의 전역적인 조명 불균형을 해결하는 데 중점을 둡니다.
- 이 값들은 이미지 특성과 원하는 결과에 따라 조절될 수 있습니다.
5. 결과 표시: 원본 이미지와 MSRCR이 적용된 이미지를 히스토그램과 함께 표시합니다.
